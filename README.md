Для бекэнда приложение автоматически находит браузер по умолчанию и на нужном порту
запускает index.html
При отправке вызывается функцию входа в систему из веб-сервиса (WSDL SOAP):
http://isapi.mekashron.com/icu-tech/icutech-test.dll
Два эндпоинта генерируют SOAP-запрос через WSDL - файл. Третий через прямой  xml soap запрос.
Все три по итогу выполняют одно действие.
Если вход в систему прошел успешно, отображается зеленое сообщение об успешном завершении вместе с информацией о сущности.
При возникновении ошибки отображается красное сообщение об ошибке.
Чтобы создать тестовую учетную запись, воспользуйтесь функцией RegisterNewCustomer на сайте
http://isapi.mekashron.com/icu-tech/icutech-test.dll?mode=default
Для фронтенда Создайн одностраничный сайт с помощью Bootstrap, который включает в себя:
Два поля для ввода (Логин и пароль). По умолчанию данные заполняются
существующей учётной записью для проверки пользователя

Порядок сборки
apt install -y dotnet-sdk-8.0
dotnet restore
dotnet build

Если развёрнуто на хостинге, то набрать
dotnet run --urls=http://*:80
